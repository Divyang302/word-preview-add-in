📄 Install the Word Add-in

  1. 📥 Click here to download the add-in file

  2. 📝 Rename the downloaded file from WordPreviewAddIn.vsto.txt to WordPreviewAddIn.vsto

  3. 🧩 Double-click the .vsto file to install the Word add-in

  4. ⚠️ If you see a security warning, follow these steps to trust the certificate:

    1. Download the certificate
    2. Double-click it and click “Install Certificate”
    3. Choose “Local Machine”
    4. Select:
      • Trusted Root Certification Authorities
      • Trusted Publishers
    5. Click “Next” and then “Finish”
